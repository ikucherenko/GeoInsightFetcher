# The Madgicx Geo Insight Fetcher.

Requirements
------------

* Python 3.6
* Requests 2.21
* Mock 2.0

Installation
------------

    $ python3 setyp.py install
   
   
 Usage
-----
Type:  

    $ python3 madgicx_geo.py -help 
    
or

    $ python3 madgicx_geo.py -h
    
    
for help


 Tests
-----

    $ python3 setup.py test
